# ClinicEMR

ClinicEMR is an open-source, procedural PHP-based Electronic Medical Records (EMR) system designed for practical use in clinics and hospitals.

The project focuses on real clinical workflows, simplicity, and reliability, with an emphasis on self-hosted and on-premise deployments.

## Features

* Patient registration and visit management (OPD, IP, Emergency)
* Clinical documentation and consultations
* Pharmacy billing and stock deduction
* Billing, invoicing, and payments
* Inventory and procurement tracking
* Role-Based Access Control (RBAC)
* Clinical and financial reporting

## Technology

* PHP (procedural)
* MySQL / MariaDB
* Linux (Debian/Ubuntu)
* Nginx or Apache
* Bootstrap and JavaScript (AJAX)

## Status

ClinicEMR is under active development.
Documentation and public demo access will be expanded over time.

## Installation

1. Clone the repository
2. Import the database schema
3. Configure `config.php`
4. Set required directory permissions
5. Access via web browser
